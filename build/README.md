# VRRTest distribution
If you want to build Linux or Windows executables yourself, you'll need to extract the latest compatible LÖVE version in this folder.

## Windows
Extract the latest portable LÖVE .zip file in `build/win`: `unzip -j -d build/win love-11.5-win64.zip`

## Linux
Extract the latest LÖVE .AppImage file in `build/lin`: `cd build/lin && /path/to/love-11.5-x86_64.AppImage --appimage-extract`
